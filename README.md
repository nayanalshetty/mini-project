#Mini Project-Scientific Calculator
This is a simple scientific calculator build using c language.
It performs operations like addition, subtraction, multiplication, division, sin, cos, tan, power, etc

##How to run
1.Open the project in any C compiler(CodeBlocks, VS Code, Dev C++).
2. Compile the code.
3. Run the program

## Features
-Basic arithmetic operations
-Scientific Functions
-Error handing
